🚀 AI-Powered Object Detection System using YOLOv8

An advanced real-time Object Detection System built using YOLOv8, OpenCV, PyTorch, and Streamlit.
The system can detect and classify multiple objects from live webcam streams and uploaded images with high speed and accuracy.

📌 Features

✅ Real-time object detection using webcam
✅ Image upload detection
✅ Bounding boxes with confidence scores
✅ Fast YOLOv8 inference
✅ Streamlit interactive web interface
✅ FPS (Frames Per Second) monitoring
✅ GPU support using CUDA
✅ Modular object-oriented architecture

🧠 Technologies Used
Python
YOLOv8 (Ultralytics)
PyTorch
OpenCV
Streamlit
NumPy
Supervision Library
